# notepad

> A small project built for learning purposes. Not intended for commercial use.

## Overview

This project consists of a frontend and a backend. The goal was to build it with minimal dependencies and focus on understanding the core concepts.

- **Frontend:** Lightweight, vanilla JS, simple styling.
- **Backend:** Node.js / Express, simple REST API.

## Tech Stack

- Frontend: HTML, CSS, JS
- Backend: Node.js, Express, MongoDB
- Optional: PM2 for process management, http-server for static serving
